# DATA STOCK – Sistema de Inventario

Proyecto final del curso de Lenguaje de Programación.  
Es un sistema de inventario básico hecho en C++ que permite registrar, listar, buscar y actualizar productos.

## Funciones principales
- Registrar producto
- Listar productos
- Calcular valor total del inventario
- Buscar producto por código
- Actualizar stock o precio
- Expansión automática del inventario (cuando se llena)

## Tecnologías
- C++
- Punteros y memoria dinámica
- Consola (Windows)
